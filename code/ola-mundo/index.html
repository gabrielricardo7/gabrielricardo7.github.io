Olá, Mundo!

Code / Por Gabriel Ricardo

“Olá Mundo” ou “Alô Mundo” é um famoso programa de computador que exibe a frase “Olá, Mundo!” (ou “Hello, World!”), comumente seguido de uma quebra de linha – com algumas variações como ausência do ponto de exclamação e letras em minúscula – no dispositivo de saída. É usado como um teste ou exemplo de código minimalista de uma linguagem de programação. Segue alguns exemplos:

print("Hello world!")
print "Hello world!"
document.write("Hello world!");
namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            System.Console.WriteLine("Hello world!");
        }
    }
}
