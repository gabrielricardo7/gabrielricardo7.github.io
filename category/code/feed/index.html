É usado como um teste ou exemplo de código minimalista de uma linguagem de programação. Segue alguns exemplos:

print("Hello world!")
print "Hello world!"
<?php
echo "Hello world!\n";
?>
document.write("Hello world!");
namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            System.Console.WriteLine("Hello world!");
        }
    }
}
