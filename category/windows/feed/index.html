Tech Bench (Banco de Tecnologia)

TechBench by WZT – é um projeto que o ajudará, sem pesquisa e registro tedioso, a baixar sistemas operacionais Windows, aplicativos do Office, pacotes de idiomas e máquinas virtuais diretamente dos servidores da Microsoft. Não armazenamos nenhum arquivo no servidor disponível pelo TechBench by WZT…

No menu de seleção:

1 – Dentre as opções, escolha: tipo -> versão -> edição -> idioma -> arquivo.

2 – Depois de selecionar tudo descrito anteriormente – à direita, um botão de Download e algumas informações são exibidas no lugar da figura:

a) Quando o arquivo foi carregado no servidor da Microsoft;

b) O tamanho do arquivo em megabytes (em bytes);

c) Número de pedidos para baixar este arquivo através do Techbench da WZT;

d) A quantidade de hash do arquivo pelo algoritmo SHA1;

e) Quando o link expira, até quando estará disponível para baixar.

Acessar site para baixar

Se preferir, pode baixar a Ferramenta de Criação de Mídia de Instalação do Windows 10 ou baixar as imagens de disco do Windows 8.1 ou do Windows 7 (arquivos ISO), através do site da Microsoft.
